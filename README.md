# flappy_bird
Clon del flappy bird escrito en python. :)

## Instrucciones
1. Generar un virtual enviroment:

   '''
   python3 -m venv ./venv
   '''
1. Activar el virtual enviroment:

   '''
   . ./venv/bin/activate
   '''
1. Ejecutar el videojuego
   '''
   python flappy_bird/flappy_bird.py
   '''
